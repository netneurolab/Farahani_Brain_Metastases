Further information:


The files in this folder are downloaded as dcm files from OSF directory of the BM study.
We have next used "dcm2niix" to convert them to nifti.
The renamed files are provided here.


The affine of BM maps and the T1w data they have provided on the OSF is different from Sym09a
So as the first step, I made them all have the same affine as the MNI space.
The new ones are saved as "_corrected".
